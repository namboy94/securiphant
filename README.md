# securiphant

securiphant is a surveillance, security and home automation solution
meant to be used on a raspberry pi. It offers
(or at least eventually should offer):

* Remote Access to video footage of cameras connected to the raspberry pi
* A voice assistant like Siri/Alexa etc.
* A display displaying various stuff
* An automated break-in detector with alerts sent via telegram

The project is currently in the planning stages. Planning takes place
[here](planning/)
